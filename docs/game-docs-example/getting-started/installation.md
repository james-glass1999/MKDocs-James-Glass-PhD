# Installation

This guide walks you through installing Unity and preparing your environment.

## Audience

**Level:** Beginner  
**Estimated time:** 10–15 minutes  

## Prerequisites

- A machine that meets Unity system requirements
- Internet connection

## Steps

1. Download and install Unity Hub
2. Open Unity Hub and sign in
3. Install the required Unity Editor version
4. Add any required modules (e.g. Windows/Mac build support)

## Verify installation

- Unity Hub opens successfully
- Editor launches without errors
