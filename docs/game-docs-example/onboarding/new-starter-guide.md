# New Starter Guide

Welcome to the Unity documentation example. This guide is designed to help you get up and running quickly with the project.

## Audience

**Level:** Beginner  
**Estimated time:** 15–20 minutes  

## Before you begin

Make sure you have:

- Unity Hub installed
- the correct Unity Editor version
- a supported IDE (Visual Studio, VS Code, or Rider)

## What you'll do

By the end of this guide, you will be able to:

- install and configure Unity
- open the project locally
- run the main scene
- understand the basic project structure

## Start here

Follow these steps in order:

1. [Installation](../getting-started/installation.md)
2. [Project Setup](../getting-started/project-setup.md)
3. [First Scene](../getting-started/first-scene.md)

## Next steps

Once you have completed the setup:

- [First Week Checklist](first-week-checklist.md)
- [GameObjects and Components](../core/gameobjects-and-components.md)

## Need help?

If you run into issues:

- check that your Unity version matches the project requirements
- review the setup steps again
- confirm all dependencies are installed
